Brand system
============

The early ``fincore`` brand direction is noir brutalist: dark, lean, structural,
and restrained. The visual language should feel like financial infrastructure,
not a trading game: concrete surfaces, hard lines, sparse accents, and quiet
authority.

Wordmark
--------

The current wordmark lives at ``docs/_static/fincore-wordmark.svg``.

It is intentionally simple:

* a near-black field
* a restrained oxblood signal line
* a lean market-signal monogram
* a condensed uppercase ``FINCORE`` wordmark
* a small uppercase infrastructure strapline

This is a starting point, not a final corporate identity. The name and font
treatment matter because the project is intended to become a recognizable
developer-facing financial data product.

Palette
-------

Noir black
    ``#0A0A0A``. Used for the deepest surfaces.

Concrete black
    ``#101010``. Used for primary documentation surfaces.

Iron
    ``#171717``. Used for secondary panels and code blocks.

Oxblood
    ``#7A1E1E``. Used for restrained command accents, links, and the monogram.

Bone
    ``#E7E0D0``. Used for primary text.

Ash
    ``#A8A299``. Used for secondary text.

Typography
----------

The provisional wordmark uses a condensed sans style. Interface text uses a
condensed/system sans stack so documentation feels direct, compact, and
operational without becoming bulky.

Design principles
-----------------

* Commanding but lean.
* Clear before decorative.
* Data-system language before trading jargon.
* Finance concepts explained for engineers.
* Every visual decision should support restraint, trust, and operational clarity.
